# 🚀 选股模型与量化交易模型 - 快速开始指南

## ✅ 功能已集成到Web界面

### 访问新功能

1. **启动应用**
   ```bash
   cd TradingAgents-CN
   source env/bin/activate  # 或激活虚拟环境
   python start_web.py
   ```

2. **打开浏览器**
   - 访问: http://localhost:8501
   - 登录系统（默认：admin/admin123）

3. **选择功能**
   - 在左侧侧边栏的"🎯 功能导航"下拉菜单中
   - 选择 **"🔍 智能选股"** 或 **"💹 量化交易"**

## 📊 智能选股 - 快速上手

### 第一步：选择策略
在侧边栏选择策略类型：
- **平衡型（推荐）**：技术面35% + 基本面35%
- **保守型**：基本面50% + 技术面25%
- **激进型**：技术面40% + 基本面30%
- **价值型**：基本面60% + 技术面20%
- **成长型**：技术面40% + 情绪20%

### 第二步：输入股票
有三种输入方式：
1. **手动输入**：每行输入一个股票代码
   ```
   000001
   600519
   002701
   ```
2. **文件上传**：上传包含股票代码列的CSV文件
3. **示例股票**：使用预设的示例股票列表

### 第三步：执行筛选
点击 **"🚀 开始智能筛选"** 按钮，等待分析完成

### 第四步：查看结果
- 查看评分分布图表
- 浏览推荐股票列表
- 查看每只股票的详细评分
- 导出结果（CSV或JSON）

## 💹 量化交易 - 快速上手

### 第一步：初始化交易器
在侧边栏配置：
- **初始资金**：默认10万（可调整）
- **交易策略**：选择4种策略之一
  - 趋势跟踪：适合趋势明显市场
  - 均值回归：适合震荡市场
  - 动量策略：适合强势市场
  - 多因子策略：适合所有市场（推荐）
- **最大持仓数**：默认5只
- **单笔风险**：默认2%

点击 **"🔄 初始化交易器"**

### 第二步：生成交易信号
1. 切换到 **"📡 信号生成"** 标签页
2. 输入股票代码（如：002701）
3. 输入当前价格（或等待自动获取）
4. 点击 **"🔍 生成交易信号"**
5. 查看信号类型（买入/卖出/持有）和强度

### 第三步：执行交易
1. 如果信号是"买入"或"卖出"
2. 确认交易价格和数量（留空自动计算）
3. 点击 **"✅ 执行 [买入/卖出]"**
4. 系统自动执行交易并管理风险

### 第四步：监控持仓
- 查看投资组合总资产和收益率
- 查看当前持仓明细
- 查看持仓盈亏情况
- 系统自动检查止损止盈

### 第五步：查看历史
切换到 **"📊 历史交易"** 标签页：
- 查看所有交易记录
- 查看累计盈亏曲线
- 查看胜率统计

## 💡 使用示例

### 示例1：筛选A股优质股票

1. 选择策略：**平衡型**
2. 输入股票：
   ```
   000001
   600519
   600036
   000002
   002701
   ```
3. 市场筛选：选择 **A股**
4. 点击筛选
5. 查看结果，选择综合评分>75的股票

### 示例2：趋势跟踪交易

1. 初始化交易器：
   - 资金：100,000
   - 策略：**趋势跟踪**
   - 最大持仓：5只
2. 选择一只股票生成信号
3. 如果出现"买入"信号，执行交易
4. 设置止损：-5%，止盈：+15%
5. 监控持仓，等待止盈或止损

## 🔍 故障排除

### 问题1：模块导入失败
**解决方案**：
```bash
cd TradingAgents-CN
source env/bin/activate
pip install -r requirements.txt
```

### 问题2：筛选结果为空
**可能原因**：
- 股票代码格式不正确
- 筛选条件过于严格
- 数据源不可用

**解决方案**：
- 检查股票代码格式（A股6位数字）
- 降低最小评分要求
- 检查数据源配置

### 问题3：量化交易信号不准确
**说明**：
- 当前版本为框架版本，部分功能需要完善
- 建议结合手动判断使用
- 建议先进行模拟交易测试
